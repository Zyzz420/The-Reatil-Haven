# Build a Complete Project Management Dashboard

This repository hosts a full-stack project management dashboard.  
The frontend is built with **Flutter**, while the backend uses **Node.js/Express + Prisma**.

## Technology Stack

- **Frontend**: Flutter (Material 3, Riverpod, GoRouter, Dio, fl_chart)
- **Backend**: Node.js with Express, Prisma (PostgreSQL ORM)
- **Database**: PostgreSQL, managed with PgAdmin
- **Cloud**: AWS components (EC2, RDS, API Gateway, Amplify, S3, Lambda, Cognito)

## Getting Started

### Prerequisites

Ensure you have the following installed:

- Git  
- Flutter SDK  
- Node.js 18+ and npm  
- PostgreSQL + PgAdmin
